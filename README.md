## 📖 프로젝트 소개
굿럭버디는 대인관계 속에서 얻은 경험과 충고를 편지로 기록하고,<br>
비슷한 고민을 가진 사람들이 그 편지를 찾아볼 수 있는 커뮤니티 서비스입니다.

**진행 기간:** 2026년 2월 2일~ 2026년 2월 28일

## 💝 핵심 기능

### 소셜 로그인
- 카카오 로그인
- 구글 로그인
### 편지 관련 기능
- 편지 작성, 수정, 삭제, 좋아요 기능
- 편지 정렬 기능
- 댓글 작성, 삭제, 좋아요 기능
### 추천 기능
- 회원 정보 기반 맞춤 편지 추천
